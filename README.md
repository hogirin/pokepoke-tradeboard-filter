# pokepoke-tradeboard-filter

GameWith「ポケモンカードゲーム ポケット」トレード掲示板の投稿一覧から、`もっと見る(XXX)` の XXX が 3 桁以上のものを自動で非表示にするブラウザ拡張です。

対象ページ:

https://gamewith.jp/pokemon-tcg-pocket/card-trades


## 機能

- トレード掲示板の各投稿にある「もっと見る(◯◯)」ボタンの数字をチェック
- 数字が 3 桁以上（100 以上）の投稿を、自動的に非表示にする
- 以下の操作後にも自動で再判定を実行
  - 一覧最下部の「もっと見る」での追加読み込み
  - タブ切り替え（ほしいカード / ゆずるカード など）
  - 「検索するカードを選択」からの絞り込み適用


## インストール方法（開発者モード）

1. このリポジトリをクローンまたは ZIP ダウンロードして展開する。

2. Chrome で `chrome://extensions` を開く。

3. 右上の「デベロッパーモード」をオンにする。

4. 「パッケージ化されていない拡張機能を読み込む」ボタンを押し、このプロジェクトのディレクトリを選択する。

5\. GameWith のトレード掲示板ページを開き、拡張が動作していることを確認する。



## 使用技術

- Manifest V3 ベースの Chrome 拡張
- content script による DOM 操作
- shadow DOM（`<gds-pokemon-card-pocket-trades-list>` の `shadowRoot`）内の要素取得
- MutationObserver による DOM 追加監視


## 設定・カスタマイズ

- 「3 桁以上」を変更したい場合は、`content.js` 内の判定:

```js
if (count < 100) return;
```

の数値を変更してください。
